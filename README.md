## ⚙️ Run Project Locally

Clone the repository:

```bash
git clone https://github.com/Akshat748-cmd/Reading-application.git
```

Go to the project folder:

```bash
cd Reading-application
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
